# CPL-20192-Team9
종합설계프로젝트2

# 과제
**1. 추진배경:**
> EDISON 온라인 고성능 컴퓨팅 시뮬레이션 플랫폼의 사용자들이 시뮬레이션에 시간이 얼마나 소요될지 알 수 없음

**2. 개발목표:**
> 웹 프레임워크와 restful api를 이용해서 시뮬레이션 예상 완료 시간을 사용자로 하여금 알 수 있게 해줌

**3. 기대효과:**
> 시뮬레이션 사용자들이 시뮬레이션이 언제 끝날지 예측할 수 있게 되면 연구효율이 증대될 수 있음
